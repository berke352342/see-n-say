# See 'N' Say Storymaker

This project is a JavaScript web application that emulates classic MattelÂ® See 'N' Say Storymaker toy.

## Description
Users clicks buttons to cycle through words and phrases. Once satisfied, they can build a complete story by combining their selections into a sentence.

## Technologies Used
- HTML
- CSS
- JavaScript

## How It Works
Each button cycles through an array of words. The "Build Story" button combines to the selected words and displays the final story on the page.
